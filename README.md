# Welcome to doiggie-repositories

[![License](https://img.shields.io/badge/License-BSD%202--Clause-orange.svg)](https://opensource.org/licenses/BSD-2-Clause)

This is a collection of data repository implementations for the [doiggie](https://github.com/ssciwr/doiggie) package.
If you use `doiggie` and you want to install all known data repository implementations, you can do so with:

```bash
python -m pip install doiggie-repositories
```

Similarly, if your library application uses `doiggie`, you may decide to opt-in into including all implementations
by depending on this package.
