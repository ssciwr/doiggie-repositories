# Welcome to pooch-repositories

[![License](https://img.shields.io/badge/License-BSD%202--Clause-orange.svg)](https://opensource.org/licenses/BSD-2-Clause)

This is a collection of data repository implementations for the [pooch-doi](https://github.com/ssciwr/pooch-doi) package.
If you use `pooch-doi` and you want to install all known data repository implementations, you can do so with:

```bash
python -m pip install pooch-repositories
```

Similarly, if your library application uses `pooch-doi`, you may decide to opt-in into including all implementations
by depending on this package.
